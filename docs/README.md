# Public policy pages (GitHub Pages)

This folder is intended to be published via **GitHub Pages** so Google Play can access a public **Privacy Policy URL**.

## Enable GitHub Pages

1. Push this repo to GitHub.
2. In GitHub repo settings → **Pages**
3. Set:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/docs`
4. Save and wait for the Pages build to complete.

## Privacy Policy URL

Once published, your privacy policy will be available at:

- `https://<your-github-username>.github.io/<repo-name>/privacy-policy`

Use that URL in Google Play Console → **App content → Privacy Policy**.
